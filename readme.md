# Notes

## Commands

```
git init                # creates a new repo
git add <filename>      # add one file to stage
git commit -m '<msg>'   # commit with inline msg
git status              # help i need an adult
git add -A              # add all to stage
git checkout -b <branchName>    # move to branch and create it
git branch              # list all branches
git checkout            # changes branches
git merge <branch>      # merges target branch into current


git push origin <branch> # Push target branch up
git pull origin <branch> # pull target branch into current branch and merge
git remote add origin <url> # Create a new blank repo on github and copy this line
git merge <branch>      # Merge Target Branch into current branch

git tag -a '<semVer Number>' -m 'msg'   # tag the current most recent commit
git push origin --tags      # push tags to your origin
```
